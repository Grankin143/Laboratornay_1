1) Установить Git

2) Провести первоначальную настройку системы контроля версий git

   git config --global user.name "Yuri"
   git config --global user.email "grankin42w@mail.ru"

3) Создать каталог для выполнения лабораторной работы(каталог проекта)

   cd :D/lab

4) Инициализировать git +

   git init

5) В каталоге проекта создать текстовый файл отчета (ФИО_ЛР1.txt в кодировке Win1251). В этом файле будет отчет по ЛР №1. Проиндексировать его.

   git add Grankin_lr1

6) Cделать первый коммит

   git commit -m "Отчет"

7) Добавить в файл отчета описание всех действий которые вам пришлось выполнить до текущего момента.

8) Проиндексировать изменения.

   git add Grankin_lr1.txt

9) Cделать второй коммит+

   git commit -m "Второй коммит"

10) При выполнении остальных заданий обязательно вести отчет о своей работе. После внесения любых изменений в файл отчета проиндексировать изжмененияи сделать коммит.

11) Продемонстрировать игнорирование , сравнение, удаление и перемещение файлов.

    mkdir git_ignore
    cd D:/lab/git_ignore
    (открыл файл Notepad++, написал *.txt)
    touch readme.txt (создаем .txt файл)
    git add .
    git status (файл readme игнорируется)
    
    сравнение
    cd D:/lab
    git diff --staged
    
    сравнения коммитов
    git log --oneline
    git diff 98d6b94 ba0fe6a

    удаление
    cd D:/lab/git_ignore
    rm t.txt
    git add t.txt

    перемещение
    cd D:/lab
    mkdir papka
    touch g.txt
    git add .
    mv g.txt/papka
    git add g.txt
    git add papka
